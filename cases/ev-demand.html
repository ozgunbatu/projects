EV Demand Forecast – Germany

← Back to Cases

Problem

Over-stocking in some regions and stockouts in others caused inefficiencies in operations and increased holding costs.

Approach

Cleaned 2.3M transaction rows in SQL, performed region-wise demand analysis, seasonal trend analysis, and simulated stock allocation.

Recommendation

Implement a region-wise inventory allocation dashboard with weekly SQL updates for the operations team.

Implementation

Power BI dashboard with interactive slider to simulate demand changes by region, enabling data-driven allocation decisions.

Results / KPIs

  • Inventory −7%
  • Forecast error −22%
  • Decision speed +15%
  • Regional allocation optimized across 6 regions
