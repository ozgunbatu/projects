EV Demand Forecast – Germany

Optimized regional inventory and forecast accuracy using SQL, Power BI & data-driven simulations.

← Back to Portfolio

Problem

Certain regions in Germany experienced both overstocking and stockouts, causing operational inefficiency. Decision-making was slow due to lack of regionally consolidated insights.

Approach & Recommendations

  • Cleaned 2.3M transaction rows using SQL and aggregated regional demand.
  • Analyzed seasonal and monthly trends to forecast EV demand per region.
  • Built interactive Power BI dashboards to simulate inventory scenarios.
  • Suggested weekly operational updates to optimize allocation across regions.

Implementation

Power BI dashboard with sliders to simulate inventory allocation changes per region. Real-time Forecast vs Actual charts and KPI counters to measure improvement.
